# EmpresaTI - Site Institucional

Projeto estático em HTML, CSS e JavaScript para publicar no GitHub Pages.

## Estrutura

```text
empresati-site/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    ├── hero-software.svg
    ├── web-system.svg
    ├── mobile-app.svg
    ├── api-integration.svg
    ├── landing-page.svg
    └── team-work.svg
```

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto para o repositório.
3. Vá em **Settings > Pages**.
4. Em **Branch**, selecione `main` e pasta `/root`.
5. Clique em **Save**.
6. Aguarde alguns minutos e acesse a URL gerada pelo GitHub Pages.

## Ajustar WhatsApp

No arquivo `script.js`, altere esta linha:

```js
const numeroWhatsApp = '5511999999999';
```

Coloque o número real com DDI e DDD, sem espaços e sem símbolos.
